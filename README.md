# iNotes Manager

Modern task and notes management web app built with React and Vite. The application provides a clean interface for organizing personal, educational, working, and general notes in one place.

## Live Demo

[View the live demo](https://my-notes-manager-sigma.vercel.app)

## Author

Made by M Umar Alam

## Features

- Organized note categories for personal, educational, working, and general tasks
- Responsive UI with a modern gradient-based design
- Authentication flow with a dedicated auth screen
- Toast notifications for user feedback
- Smooth navigation powered by React Router

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- React Router DOM
- React Toastify
- Radix UI

## Getting Started Locally

### Prerequisites

- Node.js installed on your machine
- npm or another compatible package manager

### Installation

1. Clone the repository.
2. Install dependencies:

```bash
npm install
```

3. Start the development server:

```bash
npm run dev
```

4. Open the local URL shown in the terminal.

### Build for Production

```bash
npm run build
```

### Preview the Production Build

```bash
npm run preview
```

## Project Structure

- src/components contains the UI sections and pages
- src/contextApi contains the shared task state and context logic
- src/App.jsx defines the main routes and layout
